Frequently Asked Questions

Find answers to the most common questions.

  • What is HTML? +

    HTML is the standard markup language used to create web pages. It provides the basic structure of websites.

  • What is CSS? +

    CSS controls the appearance of HTML elements including colors, layouts, spacing, fonts and animations.

  • What is JavaScript? +

    JavaScript makes websites interactive by adding dynamic content, animations, sliders, forms and much more.
